# PROTO

Proto is an image generating software (based on [Gradio](https://www.gradio.app/)).

Proto is a mix of popular approaches to design interface for AI-poweret multitask development:

* Learned from Stable Diffusion, the software is offline, but uploading features require stable internet connection.

* Learned from Midjourney, the manual tweaking is not needed, and users only need to focus on the prompts and images.

Proto has included and automated [lots of inner optimizations and quality improvements](#tech_list). Users can forget all those difficult technical parameters, and just enjoy the interaction between human and computer to "explore new mediums of thought and expanding the imaginative powers of the human species" `[1]`.

Minimal resources: 
* GPU memory - 4GB (only for Nvidia, AMD support is queued) or higher.
* RAM memory - 8GB or higher
* Free storage space - 50GB after models installation
